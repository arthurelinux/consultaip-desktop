# Consulta IP 

É um dos meus primeiros projetos com Vue.js e Electron, usando a biblioteca Vuetify, é simples mas opera com o uso do axios e entrega uma boa performace. 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
